# v1.75 (28.11.2025)

## Новое и улучшения
- Полностью скрыты технические столбцы (UUID замена, Структурированные данные) из таблицы на шаге 2.
- В таблице "Найдены чувствительные данные":
  - Столбец "Значение" переименован в "Заменяемое значение" и расширен.
  - Столбцы "№", "Источник", "Тип" стали уже, "№" — минимально возможной ширины.
- Существенно уменьшены вертикальные отступы между меню и заголовками/контентом на всех шагах (1, 2, 3) для более компактного интерфейса.
- Исправлены баги с placeholder/пустыми UUID, теперь всегда генерируются валидные значения для всех замен.
- Гарантирована сквозная анонимизация: если выбрано значение из текста, оно заменяется и в метаданных (core.xml/app.xml/custom.xml), и наоборот.
